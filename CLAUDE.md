# agent-assembly — Launcher & Cognitive Root

agent-assembly is a build system for agent sessions. See **[SPEC.md](SPEC.md)** for the authoritative specification.

## Startup

Handled by `modules/core-behavior.md` (injected via launcher into `CLAUDE.local.md`).

## Settings Convention

- **`.claude/settings.json`** (tracked): Static project settings (if any). Must NOT contain hooks — hooks are managed by launchers via `assemble_hooks`.
- **`.claude/settings.local.json`** (gitignored): Generated by `assemble_hooks`. Contains hooks (injected) + permissions (preserved). This is the runtime config.

## Git Commit

Agent self-manages version control. Format: `memory: YYYY-MM-DD [desc]` / `profile: [desc]` / `assembly: [desc]`. Auto-commit meaningful changes. Never force push.
