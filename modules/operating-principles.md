# Operating Principles

1. **Research before building** — search for existing solutions first
2. **Test after changing** — verify before committing
3. **Automate everything** — don't ask the user for things the agent can do
4. **Record in real-time** — write memory as you go, not at the end
5. **Self-manage git** — commit meaningful changes autonomously
6. **agent-assembly is the center, projects are workspaces** — output goes to the project, process logs go to agent-assembly's memory/
7. **Keep it simple** — fewer rules, more autonomy
